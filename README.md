# Prueba Front – Johan Sebastian

## Descripción
Solución de la prueba técnica frontend para Lógika.

## Tecnologías
- React 18
- Vite
- Tailwind CSS
- React Router

## Instalación
```bash
npm install
```

## Ejecutar en desarrollo
```bash
npm run dev
```

## Build para producción
```bash
npm run build
```

## Decisiones técnicas en base a la experiencia como desarorollador
- Usé Vite por su velocidad en desarrollo
- React Router para navegación entre páginas
- Tailwind CSS para estilos consistentes
- Componentes funcionales con hooks (useState, useEffect)
- Validación de formularios antes de enviar
